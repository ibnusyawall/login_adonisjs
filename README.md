# login_adonisjs
simple login authentication with adonisjs framework

# install
```
$ npm i -g @adonisjs/cli
$ git clone https://github.com/ibnusyawall/login_adonisjs
$ cd login_adonisjs
$ npm i
$ adonis migration:run
$ adonis seed
$ adonis serve --dev
```
